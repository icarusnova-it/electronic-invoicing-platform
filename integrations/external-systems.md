# External Systems Integration

> **Icarus Nova** | Integration with ERP systems and business applications.

## Overview

The platform integrates with external systems (ERPs, business applications) via REST APIs and webhooks.

## Integration Methods

### REST API

**Endpoints:**
- Invoice creation
- Status checking
- Document retrieval
- Webhook configuration

### Webhooks

**Events:**
- Invoice authorized
- Invoice rejected
- Authorization received
- Status changes
